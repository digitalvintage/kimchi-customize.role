Role Name
=========

An Ansible role for Wok/Kimchi customization. Also fixes some visual bugs

Requirements
------------

Installed Kimchi/Wok. Tested on version 2.5.0

Role Variables
--------------

wok_customize:
  system_title: Wok
  system_logo: default
  logo_padding: 60px
  show_plugin_info: false
  show_wok_version: true
